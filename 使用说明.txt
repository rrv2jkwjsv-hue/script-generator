📖 脚本生成工具使用说明
======================

⚠️ 重要：由于浏览器安全限制，直接双击打开HTML文件会导致API调用失败。

✅ 正确使用方法（3选1）：

方法1：使用本地代理服务器（推荐）
--------------------------------
1. 确保已安装 Node.js（访问 nodejs.org 下载）

2. 打开命令行，进入文件所在目录：
   cd "C:\Users\xchu5\Desktop\AI探索"

3. 运行代理服务器：
   node proxy-server.js

4. 看到提示"✅ CORS代理服务器运行在 http://localhost:8080"后

5. 双击打开 脚本生成工具.html 即可使用

6. 使用完毕后，在命令行按 Ctrl+C 停止服务器


方法2：使用Python HTTP服务器
----------------------------
1. 打开命令行，进入文件所在目录：
   cd "C:\Users\xchu5\Desktop\AI探索"

2. 运行HTTP服务器：
   python -m http.server 8000

3. 打开浏览器访问：
   http://localhost:8000/脚本生成工具.html


方法3：部署到Netlify（分享给他人）
----------------------------------
1. 访问 https://app.netlify.com/drop

2. 把 脚本生成工具.html 拖到页面中

3. 获得网址后分享给朋友即可


📝 注意事项：
- 每个使用者需要自己的Claude API Key
- API Key获取：https://console.anthropic.com
- 生成时间：完整脚本约40-60秒
- 网络要求：需要能访问Claude API

💡 推荐：
- 自己使用 → 方法1（本地代理）
- 分享他人 → 方法3（Netlify部署）
